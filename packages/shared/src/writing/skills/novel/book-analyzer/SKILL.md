# Book Analyzer

Adapted for Craft Agent from Claude-Book concepts. Source: https://github.com/ThomasHoussin/Claude-Book

Extract evidence-backed writing bible files from source texts into `analysis/output/<source-slug>/`. Preserve quotes, distinguish facts from interpretation, and avoid inventing missing details.
