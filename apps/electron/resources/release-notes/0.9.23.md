# 0.9.23

## 登录与发布配置

- 将桌面端默认 auth broker 收敛到 `https://storyflow-auth.zjding.com`，避免新用户继续命中旧的 `workers.dev` 地址。
- 增加已安装客户端的 `client-auth.json` broker 覆盖通道，旧包在 broker 迁移或网络受限时可以自救。
- 收敛本地 dotenv 分层：`electron:dev` 使用 `.env.local > .env.dev > .env`，构建发布路径使用 `.env.local > .env`，且都不会覆盖显式 shell/CI 环境变量。
- 本地 unsigned macOS 打包会显式关闭 app 和 DMG 签名，避免无证书环境误触发 electron-builder 签名。
