# Storyflow Promo Video

45 秒 Storyflow 产品宣传视频的 HyperFrames 工程。

## Files

- `AGENTS.md` - HyperFrames project instructions.
- `CLAUDE.md` - HyperFrames scaffold companion instructions.
- `DESIGN.md` - 视频视觉规范。
- `assets/` - 从本地录屏裁出的轻量 MP4 片段。
- `hyperframes.json` - HyperFrames 项目配置。
- `index.html` - HyperFrames 主合成。
- `meta.json` - HyperFrames 项目元数据。
- `package.json` - 本地预览、检查、渲染脚本。
- `renders/` - 渲染输出目录。
